# ClawTVOS（cto.site）

静态官网，托管 **GitHub Pages**，域名 **clawtvos.com**。

- 站点：<https://clawtvos.com>
- 仓库：<https://github.com/zengxh59/cto.site>
- 发布：推送到 `main` 根目录即更新；DNS 备忘见 `spaceship-dns-records.txt`。

**Chrome 仍提示「不安全」时**：确认地址栏是 **`https://`**（不要用 `http://` 书签）；试无痕窗口或清除该站数据；排查本机代理 / 安全软件是否截 HTTPS。

**注意**：GitHub Free 下若把仓库设为 **Private**，公开 Pages 会异常；需保持 **Public** 或升级套餐 / 另建公开发布仓。
